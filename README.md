## 如何使用？
1.安装依赖库  
```python
pip install pywin32
```
2.clone本项目到本地 
```
git clone https://github.com/Quandong-Zhang/desktop-wallpaper-engine-new.git
```
3.运行main.py,在弹出的对话框中点击“启动壁纸”，并选择一个你想作为壁纸的视频文件。  
4.enjoy it!

## 关于win11多桌面兼容功能（会耗费较多性能）  
在win11下使用多桌面功能时请点击该按钮（点一下就好）
